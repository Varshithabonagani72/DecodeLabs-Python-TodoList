# DecodeLabs Python To-Do List

## Project Overview

• This project is a simple Python To-Do List application developed during my internship at DecodeLabs

• The application helps users manage daily tasks using Python programming concepts

• This project is created using basic Python concepts for beginners

## Features

• Add new tasks

• View all tasks

• Remove completed tasks

• Simple and easy command line interface

## Technologies Used

• Python

## Project File

• todo.py – Main Python program

## How to Run the Project

• Install Python on your system

• Open Command Prompt or PowerShell

• Navigate to the project folder

• Run the command:

python todo.py
